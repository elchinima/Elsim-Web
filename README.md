## 🌌 About

Official static website for **Elsim**, a software studio from **Baku, Azerbaijan**.

This single-page website showcases products, portfolio, and contact channels with a clean dark aesthetic and motion-focused UI.

## ✨ Highlights

- 🎯 Product showcase: Shapix, Dev Helper, ECode
- 🌍 Multilingual interface: English, Russian, Azerbaijani
- 📱 Responsive layout for desktop and mobile
- ⚙️ Fully static project (no build tools required)

<details>
    <summary><strong>🎬 Visual Effects Included</strong></summary>

- Loading intro animation
- Smooth scroll behavior
- Custom interactive cursor
- On-scroll reveal animations
- Marquee motion strip
- Sticky navbar transition on scroll

</details>

| Layer | Technology |

        `-- Scripts
                `-- index.js

## 🌍 Localization

Translations are stored in `Assets/Scripts/index.js` inside the `T` object.

Supported languages:

| Code | Language |
|---|---|
| `en` | English |
| `ru` | Russian |
| `az` | Azerbaijani |

Selected language is saved in browser local storage.
